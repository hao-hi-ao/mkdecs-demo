概念参考

第1章 玩具总动员

atom

一个字符串（不能以'('和')'左右圆括号开头）或一个数字 

`atom
`turkey
`1492
`u
`*abc$
10
12

list

用括号括起来的atom集合或list集合或两者兼有 

`(atom)
`(atom turkey or)
`((atom turkey) or)

null list/empty list

括号内没有元素 

`()

S-expression

atom和list都是S-expression 

`xyz
`(x y z)
`((x y) z)
